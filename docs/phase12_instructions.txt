Phase 12 — 최종 테스트 및 한글 호환성 검증

test_notice.json으로 전체 문서 생성 후 DoD 체크리스트 기준으로 결과 보고해줘.

자동 검증 가능한 항목은 코드로 확인하고,
나머지는 내가 한글에서 직접 확인할게.

최종 결과물:
- final_test.hwpx (test_notice.json 기반)
- final_test_md.hwpx (test_sample.md 기반 — Phase 11 마크다운 파서 검증)

DoD 체크리스트:
[ ] 한글 2020 이상에서 파일 오류 없이 열림
[ ] 폰트명 적용됨 (HY헤드라인M, 맑은 고딕)
[ ] 줄간격 160% 반영
[ ] 표 셀 여백 좌우 3mm / 상하 2mm 반영
[ ] 공문서 불릿 5단계 계층 정상 출력
[ ] 페이지 여백 상하 10mm / 좌우 20mm 적용
[ ] 표가 페이지 중간에서 잘리지 않음
[ ] 섹션 제목이 페이지 맨 아래에 혼자 남지 않음
[ ] 소제목이 다음 내용과 항상 같은 페이지에 위치
[ ] 고아/과부 줄 발생 없음
[ ] PNG/JPG 이미지 지정 크기/정렬로 삽입됨
[ ] 도형이 HWPX 네이티브 벡터로 출력됨 (한글에서 클릭 편집 가능)
[ ] 차트가 OOXML 네이티브 벡터로 출력됨 (한글에서 더블클릭 데이터 편집 가능)
[ ] SVG가 네이티브 도형으로 변환됨
[ ] 강조 텍스트박스 테두리/배경색 정상 출력
[ ] 도형/차트 포함 문서 5초 이내 생성
[ ] 생성 후 한글에서 재저장 시 파일 손상 없음
