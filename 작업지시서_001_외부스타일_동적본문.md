## 배경

외부 스타일(`--style 외부`)은 현재 템플릿 고정 슬롯 방식이라 슬롯 수 초과 및 계층 구조 소실 문제가 있음. 본문(파트2)을 동적으로 생성하는 방식으로 교체함.

---

## 확정된 방향

| 영역 | 처리 방식 |
|------|-----------|
| 파트0 (표지) | 템플릿 보존, 텍스트만 치환 |
| 파트1 (목차) | 템플릿 보존 |
| 파트2 (본문) | 통째로 제거 후 동적 재생성 |
| 섹션바 | 원형 복제 → 제목 텍스트 교체 |
| 슬롯 단락 | 원형 복제 → 텍스트 교체, 개수 제한 없음 |
| 색상 | header.xml 값을 CLI 옵션으로 오버라이드 |
| 표지 장식 | 템플릿 장식 유지, 색상만 color_main 연동 |

## 구현 내용

### 1. split_sections()
(section0.xml → 파트0/1/2 분리)  
`pageBreak="1"` 기준으로 3개 영역 분리. 반환: `(part0_xml, part1_xml, part2_xml)`

### 2. extract_prototypes()
(파트2 → 슬롯 원형 5종 추출)  
추출 대상:
- `section_bar`: 섹션바 테이블 (3열 1행, treatAsChar="1")
- `slot_square`: □ 단락 (paraPrIDRef=28)
- `slot_circle`: ○ 단락 (paraPrIDRef=29)
- `slot_dash`: ― 단락 (paraPrIDRef=30, charPrIDRef=18)
- `slot_note`: ※ 단락 (paraPrIDRef=30, charPrIDRef=20)

### 3. parse_markdown_sections()
(마크다운 → 섹션별 계층 구조 파싱)  
현재 플랫 수집 방식을 섹션별 수집으로 변경. 반환 형식:
```python
[
  {
    "title": "Ⅰ. 추진 배경",
    "items": [
      {"type": "square", "text": "항목1", "children": [
        {"type": "circle", "text": "하위항목", "children": [
          {"type": "dash", "text": "세부항목"}
        ]}
      ]},
      {"type": "note", "text": "주석내용"}
    ]
  }
]
```

### 4. build_part2()
(섹션 데이터 + 원형 → 새 파트2 XML 조립)  
- 섹션바 원형 복제 → 로마숫자 + 제목 텍스트 교체  
- 슬롯 원형 복제 → 텍스트 교체  
- 계층 순서 유지: □ → ○ → ―  
- `copy.deepcopy` 사용

### 5. apply_colors()
(header.xml 색상 오버라이드)  
교체 대상:

| 대상 | 적용 옵션 |
|------|-----------|
| borderFill ID=7,9 faceColor | `--color-main` |
| borderFill ID=9 border color | `--color-main` |
| charPr ID=0 (□기호) textColor | `--color-sub` |
| charPr ID=20 (※) textColor | `--color-accent` |

### 6. replace_cover_text()
(파트0 표지 텍스트 치환)  
charPrIDRef=22 인 run → 제목, 기관명 교체. 날짜 텍스트 교체.

### 7. assemble()
(파트0 + 파트1 + 새 파트2 → 최종 section0.xml)

## CLI 변경

`cli.py`에 아래 옵션 추가:

```
--color-main     (기본값: 1F4E79)
--color-sub      (기본값: 2E75B6)
--color-accent   (기본값: C00000)
```

`--style 외부` 실행 시 기존 슬롯 치환 방식 → `dynamic_builder.py` 호출로 교체.

- XML 파싱: `xml.etree.ElementTree` 사용 (외부 라이브러리 없이)

- 원형 복제 시 `copy.deepcopy` 사용
