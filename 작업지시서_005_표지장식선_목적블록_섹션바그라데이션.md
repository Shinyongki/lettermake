## 작업 5-1 — 표지 장식선 교체

### 현재 상태
표지(파트0) 테이블 행0/행2에 borderFill=7(#193AAA), 8(#A0B4E6) 색상 바가 있음.

### 목표
색상 바 대신 `hwpx_generator`의 `doc_title_top.png` / `doc_title_bottom.png`를
표지 제목 위아래에 삽입.

### 구현 순서

1. PNG 파일 경로 확인
   - `hwpx_generator/elements/` 또는 `hwpx_generator/assets/` 에서
     `doc_title_top.png`, `doc_title_bottom.png` 위치 확인

2. hwpx ZIP에 PNG 추가
   - `BinData/image2.png` (top), `BinData/image3.png` (bottom) 으로 추가
   - `Contents/content.hpf`에 BinData 항목 등록

3. 파트0 표지 테이블 수정
   - 행0 (상단 색상 바 셀): 배경 제거, `<hp:pic>` 으로 top PNG 삽입
   - 행2 (하단 색상 바 셀): 배경 제거, `<hp:pic>` 으로 bottom PNG 삽입
   - `<hp:pic>` 구조는 `doc_title.py`의 `_render_decoration_pic()` 참조

4. 이미지 폭/위치
   - 폭: 외부 스타일 본문 영역 폭 (page_width - margin_left - margin_right)
   - 높이: 708 hwp unit (약 2.5mm) 고정
   - `binaryItemIDRef`: image2, image3 (기존 image1과 충돌 없게)

## 작업 5-2 — 목적 블록 추가

### 현재 상태
마크다운 `목적:` 라인이 외부 스타일에서 무시됨.

### 목표
표지(파트0) 하단에 테두리 박스 형태로 목적 텍스트 삽입.

아래와 같은 형태:
```
┌─────────────────────────────────────┐
│ 부서별 핵심 과제를 정리하고 효율적인 │
│ 업무 추진 방향을 공유하기 위함       │
└─────────────────────────────────────┘
```

### 구현 순서

1. `parse_markdown_sections()`에서 `목적:` 라인 파싱 추가
   - 반환값에 `purpose` 키로 저장
   ```python
   { "purpose": "부서별 핵심 과제를...", "sections": [...] }
   ```

2. `replace_cover_text()`에서 목적 블록 삽입
   - 파트0 테이블 아래에 1행 1열 테이블 추가
   - 테두리: SOLID, `#000000`, 0.5mm
   - 배경: 없음 (투명)
   - 폰트: charPrIDRef=5 (본문 폰트)
   - 셀 여백: 상하 3mm, 좌우 5mm

## 작업 5-3 — 섹션바 그라데이션 적용

### 현재 상태
섹션바 좌측 셀: `<hc:winBrush faceColor="#1F4E79">` (단색)
섹션바 우측 셀: `<hc:winBrush faceColor="#F2F2F2">` (단색)

### 목표
단색 → 그라데이션으로 교체. 기존 방식 문서의 섹션바와 동일한 효과.

### 구현 순서

1. hwpx 그라데이션 XML 구조 파악
   - `hwpx_generator` 코드에서 그라데이션 생성 부분 확인
   - 없으면 hwpx 스펙 기준으로 `<hc:gradation>` 태그 직접 작성

2. 그라데이션 적용 방식
   - `<hc:winBrush>` → `<hc:gradation>` 으로 교체
   - 좌측 셀: `color_main` → 약간 밝은 계열로 좌→우 그라데이션
   - 우측 셀: 연한 회색 → 흰색 방향 그라데이션

3. `apply_colors()`에서 borderFill 수정 시 그라데이션도 함께 적용

### hwpx gradation 태그 구조 참고
```xml
<hc:gradation type="LINEAR" angle="0" border="0" center-x="50" center-y="50">
  <hc:gradationColor startColor="#1F4E79" endColor="#2E75B6"/>
</hc:gradation>
```
정확한 태그명/속성은 hwpx 스펙 또는 기존 파일에서 확인 후 사용.
