# cli.py 수정 지시문

기존 cli.py에 --style, --org 옵션을 추가해줘.

## 프로젝트 구조
D:\Project2\문서작성자\
├── cli.py                          ← 수정 대상
├── hwpx_generator\                 ← 기존 라이브러리
└── gonggong_hwpxskills-main\
    ├── assets\report-template.hwpx
    └── scripts\fix_namespaces.py

## 추가할 옵션 2개

--style (-s)
  선택값: "기존" | "외부"
  기본값: "기존" (생략 시 hwpx_generator 동작 그대로)
  - "기존": 기존 hwpx_generator 방식
  - "외부": gonggong_skill 보고서 템플릿 방식

--org
  보고서 기관명 (--style 외부 일 때만 사용)
  기본값: "" (공백)

## 사용 예시
python cli.py -i report.md -o report.hwpx --style 외부 --org 경기도교육청
python cli.py -i report.md -o report.hwpx --style 외부   # 기관명 공백
python cli.py -i notice.md -o notice.hwpx                # 기존 방식 (생략)
python cli.py -i notice.md -o notice.hwpx --style 기존   # 기존 방식 (명시)

### 마크다운 파싱 규칙
# 제목       → 표지 제목, 본문 제목
## Ⅰ. 섹션  → 섹션 바 제목 (최대 4개)
- 항목       → □ 슬롯 순차 치환 (최대 8개)
  - 항목     → ○ 슬롯 순차 치환 (최대 8개)
    - 항목   → ― 슬롯 순차 치환 (최대 8개)
※ 주석       → ※ 슬롯 순차 치환 (최대 7개)

### 템플릿 플레이스홀더 매핑
"브라더 공기관"                          → --org 값 (공백이면 빈 문자열)
"기본 보고서 양식"                        → # 제목
"2024. 5. 23."                          → 오늘 날짜 자동 (형식: 2026. 3. 5.)
"제 목"                                 → # 제목
섹션 바 기존 텍스트 4개                   → ## 섹션 제목 순서대로
"헤드라인M 폰트 16포인트(문단 위 15)"     → - 항목 순차
"  ○ 휴면명조 15포인트(문단위 10)"       → -- 항목 순차
"   ― 휴면명조 15포인트(문단 위 6)"      → --- 항목 순차
"     ※ 중고딕 13포인트(문단 위 3)"      → ※ 항목 순차

### 섹션 바 기존 텍스트 4개 (순서대로 치환)
" 추진 배경"
" 현황 및 문제점"

### 슬롯 초과 시
경고 출력 후 슬롯 수만큼만 삽입
예: Warning: □ 항목이 8개를 초과합니다. 8개까지만 삽입됩니다.

### 후처리 (필수)

## 주의사항
- 기존 --preset, --no-page-flow 옵션 동작 완전히 유지
- --style 기존 또는 생략 시 기존 코드 한 줄도 바뀌지 않아야 함
- fix_namespaces.py 는 exec() 아닌 subprocess.run() 으로 실행
